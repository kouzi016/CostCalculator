# 毎月計算機アプリ（経費管理）

## 📱 アプリ概要
毎月の経費を簡単に計算・管理し、**お父さんに送信するため**の経費計算アプリです。各カテゴリの支出を入力すると自動で合計金額を算出します。

---

## ⚡ 現在実装済みの機能

### 💰 経費カテゴリ管理

#### 🚌 バス代
- **660円固定**（1クリックで追加）

#### 🗣️ 英会話
使用頻度に応じた料金設定
- **12回未満**：0円
- **12回以上**：3,500円  
- **20回以上**：7,000円

#### 🍽️ 食費
週単位での選択制
- **週1回**：2,000円
- **週2回**：4,000円
- **週3回**：6,000円
- **週4回**：8,000円
- **週5回**：10,000円

#### 🚲 駐輪場代
- **1,100円固定**

#### 🚗 自動車学校代
- **8,000円**（支出として計算）

#### ➕ その他
- **自由入力**（カスタム金額）

---

### 📊 計算・表示機能

| 機能 | 詳細 |
|------|------|
| **リアルタイム計算** | 各項目の合計を自動算出 |
| **詳細表示** | カテゴリ別の内訳を一覧表示 |
| **コピー機能** | 計算結果をテキスト選択・コピー可能 |
| **リセット機能** | 全項目を一括リセット |

---

### 🔔 通知機能

- 📤 経費入力とお父さんへの送信リマインダー
